How to embed svg icons in html page

Follow the steps to get the icons :
  • go to icomoon.io website.
  • select the button Icomoon App
  • if not yet selected before, select the icon pack Entypo+ in the library
  • select the desired icons
  • select the button Generate SVG & more and download the zip file

From the unzipped file, keep only the SVG folder, the demo.html file and the symbols-defs.svg file. Rename symbol-defs.svg to sprite.svg. Place the folder SVG and the files demo.html and sprite.svg into your img file. Finally, place the following code where appropriated :  <svg class="linkedin-icon">
    <use xlink:href="img/sprite.svg#icon-linkedin">
    </use>
 </svg>
